# Rick and Morty-pedia

Search any character of the Rick and Morty universe in this tiny Vue app!

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
